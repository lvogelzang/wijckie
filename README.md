# wijckie

Personal start page

### How it's built

- Setup front-end:  
  `yarn create vite front-end --template react-ts`  
  `yarn install`
- Add Prettier, add configuration, write changes:
  `yarn add --dev prettier`
  `nano .prettierrc`
  `yarn prettier --write 'src/**/*.{js,ts,tsx}'`
- Add SASS, Bootstrap
  `yarn add sass`
  `yarn add bootstrap react-bootstrap`
- Run application
  `yarn run dev`
